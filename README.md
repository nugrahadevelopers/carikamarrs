## Cari Kamar RS

Aplikasi untuk melihat ketersediaan kamar Rumah Sakit, dibangun menggunakan:

- Laravel 8
- Livewire
- Tailwind CSS
- API from [satyawikananda](https://github.com/satyawikananda/rs-bed-covid-indo-api/)
